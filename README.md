# Telegram бот для заявок по авто и тюнингу

Бот помогает собрать «тёплую» заявку на услуги:

- привоз авто из Азии под ключ;
- тюнинг и доработки;
- резина и расходники;
- детейлинг и подготовка;
- консультации по подбору и обслуживанию.

## Запуск

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Укажите переменные окружения:

- `BOT_TOKEN` — токен Telegram-бота.
- `ADMIN_CHAT_ID` — (необязательно) numeric ID администратора для доступа к `/leads` и получения заявок.

3. Запустите бота:

```bash
python bot.py
```

Заявки сохраняются построчно в `leads.jsonl`. Если указан `ADMIN_CHAT_ID`, сводка заявки отправляется администратору сразу после оформления.
